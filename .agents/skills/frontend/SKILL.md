# Frontend Skill Guidelines
- **Framework**: Next.js
- **Styling**: Standard CSS (`.css` files) or CSS modules. Avoid heavy reliance on Tailwind or external UI libraries unless specifically requested.
- **Structure**: Emphasize clean, semantic HTML inside React components (JSX).
